  │  Pin 2 ── Phase B (120°) PWM out ──────┼──> [Phase B RC Filter]
  │  Pin 4 ── Phase C (240°) PWM out ──────┼──> [Phase C RC Filter]

RC LOW-PASS FILTER — PHASE A (replicate for B and C)
=====================================================

  100kHz PWM in (0–3.3V)
        |
       [R1 1kΩ]
        |
        +──────────────[R2 1kΩ]──────────────────> to X9C104 Vw (wiper in)
        |                                  |
      [C1 100nF]                        [C2 100nF]
        |                                  |
       SGND                              SGND

  Pole 1: f = 1/(2π × 1kΩ × 100nF) ≈ 1.6kHz
  Pole 2: f ≈ 1.6kHz
  Attenuation at 100kHz: ~40dB (two poles combined)


X9C104 DIGITAL POTENTIOMETER — AMPLITUDE CONTROL (shared, one per channel)
===========================================================================

                        +5V (or +3.3V)
                            |
  ┌─────────────────────────────────────────────┐
  │                  X9C104                      │
  │                                              │
  │  Pin 1 (INC)  ◄── Pico Pin 6                │
  │  Pin 2 (U/D)  ◄── Pico Pin 7                │
  │  Pin 3 (Vss)  ── SGND                        │
  │  Pin 4 (Vw)   ── wiper ──────────────────────┼──> to LM358 input
  │  Pin 5 (Rh)   ── [R3 1kΩ to SGND]           │    (voltage divider tap)
  │  Pin 6 (Rl)   ── RC filter output            │
  │  Pin 7 (Vdd)  ── +5V                         │
  │  Pin 8 (CS)   ◄── Pico Pin 8                 │
  └─────────────────────────────────────────────┘

  X9C104 acts as voltage divider:
    Rl = RC filter output (full signal)
    Rh = bottom of divider (to GND via R3)
    Vw = tapped output to amplifier input
  Turn wiper up/down via Pico to set fault current level (2×, 6×, 10×, etc.)


LM358 INPUT BUFFER + BIAS DRIVER — PHASE A
===========================================

  Signal from X9C104 wiper
        |
        | (+) non-inverting input
     [LM358 — U1A]                     +12V ──> Pin 8 (Vcc)
        | output                        GND ──> Pin 4 (GND)
        |
        +──────────────────────────────────── feedback (unity gain)
        |  (−) inverting input tied to output = voltage follower
        |
        +──────────[R4 220Ω]──────────────────────────────────> Q1 Gate (IRF540N)
        |
       [D1 1N4148]  (anode up)
        |
       [D2 1N4148]  (anode up)
        |
       [R5 trim pot 10kΩ] ── adjust quiescent current
        |
        +──────────[R6 220Ω]──────────────────────────────────> Q2 Gate (IRF9540N)
        |
       SGND

  D1 + D2 forward drops (~1.2V total) set the Class AB gate-gate offset.
  Trim pot fine-tunes quiescent current to eliminate crossover distortion.

CURRENT SENSE (OPTIONAL) — PHASE A
====================================

  OUTPUT Phase A
        |
      [R_sense 0.05Ω 5W]    ← insert in series with output lead
        |
      Relay CT Input A

  Voltage across R_sense = I × 0.05Ω
  At 10A: V = 0.5V  →  measure with DMM or feed back to op-amp for closed-loop control

HEAT SINK REQUIREMENT
======================

  Q1 (IRF540N)  : P_diss = V_ds × I = ~2–4V × 10A = 20–40W  → TO-220 heat sink required
  Q2 (IRF9540N) : same estimate                               → TO-220 heat sink required


FULL SYSTEM — THREE PHASES
===========================

  Pico Pin 0 ──[RC Filter A]──[X9C104 A]──[LM358 A]──[IRF540N/IRF9540N A]──> Relay CT A
  Pico Pin 2 ──[RC Filter B]──[X9C104 B]──[LM358 B]──[IRF540N/IRF9540N B]──> Relay CT B
  Pico Pin 4 ──[RC Filter C]──[X9C104 C]──[LM358 C]──[IRF540N/IRF9540N C]──> Relay CT C
                                   |
                         Pico Pins 6/7/8 (shared INC/UD/CS)
                         (wire all three X9C104s to same Pico pins
                          and use separate CS lines per channel if
                          independent amplitude control is needed)

  All SGND and PGND tied together at one star ground point.
  +24V, +12V rails shared across all three channels.


GROUND NOTES
=============

  SGND (signal ground) — Pico GND, LM358 GND, X9C104 Vss, RC filter returns
  PGND (power ground)  — MOSFET source returns, relay coil return, supply GND
  Star ground: join SGND and PGND at a single point near the power supply to
  prevent high-current MOSFET switching from coupling noise into the signal path.
